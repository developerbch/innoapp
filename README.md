# innoapp

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Log in
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [V] Create account
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
