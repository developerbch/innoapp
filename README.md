# innoapp

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [V] Create account
- [V] Request Secret
- [V] Confirm Secret (Login)
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
